Na początku należy pobrać node_modules za pomoca npm insatll

Jezeli chcemy zmienic adres serwera, trzeba zmienic jego ip w pliku app.js [zmienna SERVER_IP na samej gorze pliku], następnie nalezy uruchomić aplikacje z wykorzystaniem node.js wpisując w terminal node app.js

Lub skompliowac aplikacje za pomoca pkg package.json
